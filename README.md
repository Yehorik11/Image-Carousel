# Image Carousel Experience

A minimalist, high-performance image gallery built with React and TypeScript. This project focuses on elegant typography, smooth transitions, and a clean user interface inspired by premium digital editorial designs.

## ✨ Features

- **Infinite Loop Carousel:** Seamlessly browse through images with smooth CSS transitions and infinite loop logic.
- **Interactive Selection:** Toggle image selection with a simple click on any slide.
- **Dynamic Selection Gallery:** A dedicated "Selected Images" section that tracks your choices and provides direct links to sources.
- **Responsive Design:** Fully optimized for all screen sizes, from mobile devices to ultra-wide monitors.
- **Smart Loading & Error Handling:** Integrated custom spinner for handling asynchronous data fetching and API error states.
- **Clean Architecture:** Built using functional components, custom hooks for logic separation, and CSS Modules for style isolation.

## 🛠 Tech Stack

⚛️ React 18 — UI library
📘 TypeScript — type safety & DX
🎨 CSS Modules — scoped styling
🪝 Custom Hooks — reusable logic
🔄 TanStack Query — data fetching & caching

## 🚀 Getting Started

Follow these steps to get the project up and running on your local machine:

### 1. Clone the repository

```bash
git clone https://github.com/Yehorik11/Image-Carousel.git
```

### 2. Install dependencies

### 3. Run the development server
